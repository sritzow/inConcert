# inConcert
inConcert - Project Team Dashboard
The users of this program would be a team working on a project, software development or otherwise. It would be an access point for a user to create, view, and edit:
	
	project
		-goals / stories
		-milestones
		-tasks
		-progress
	
	team member
		-tasks
		-availablity
		-correspondence
		-billable hours

	calendar 	
		-deadlines
		-meetings
		-correspondence

	correspondence
		-group
		-individual
